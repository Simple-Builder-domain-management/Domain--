### Simple Builder domains
Get a free subdomain like `yourname.bdr.uk.to`

#### How we append domains
| Process | Action |
| ------- | ------ |
| You -> | Fill out a form |
| Support team -> | Review email, creates support ticket |
| Support team -> | Checks if you qualify |
| Support team -> | Sends request to software engineer team |
| Software Engineer team -> | Sets up domain using [afraid.org](https://freedns.afraid.org) |
| Software Engineer team -> | Creates a GH repo "Domain--{{domain_name}}" |
| Software Engineer team -> | Adds domain and TLS certificate |
| Software Engineer team -> | Sends "completed" request to support team (with ticket number) |
| Support team -> | Sends a success/fail email back to you |
